## 二、类型化技巧

### 修仙/玄幻
- **力量体系**：层级清晰（9级/12级），每级有质变表现
- **秘境节奏**：探索→危险→机缘→突破→回归展示
- **宗门政治**：长老会博弈、外门内门核心弟子阶层
- **天材地宝**：有来源限制和争夺逻辑

### 都市/现实
- **身份反差**：表面普通/实际背景深厚
- **商战逻辑**：需基本合理，避免"一个亿小目标"式脱节
- **社会话题**：可融入热议话题增加共鸣，但不要说教

### 宫斗/宅斗
- **情报战**：谁知道什么、不知道什么，要精确管理
- **计中计**：表面被套路→其实是反套路→再反转
- **棋子觉醒**：被利用的角色意识到真相后的选择
- **同盟瓦解**：利益分化导致阵营重组

### 穿越/重生
- **蝴蝶效应**：改变一件事引发连锁反应
- **知识变现**：利用未来知识/现代知识获利
- **时间压力**：已知的灾难节点倒计时
- **身份维持**：伪装原身份的紧张感

### 甜宠/言情
- **心跳时刻设计**：不经意的身体接触、替对方挡危险、雨中/雪中场景
- **吃醋桥段**：第三者出现时的微反应
- **甜度梯度**：牵手→拥抱→亲吻，每个阶段有铺垫
- **男主具象宠法**：不说"他很宠她"，写"他提前把她杯子里的冰块捞出来"

## 三、爽点设计框架

### 爽感公式
**爽感 = 压抑时长 × 反差幅度 × 旁观者反应强度**

### 压抑→爆发设计模板
```
第N章：主角被[具体打压事件]，[围观者]嘲笑/轻视
第N+1章：持续压抑，增加新的不利因素
第N+2章：转折契机出现（灵感/突破/外援/揭秘）
第N+3章：主角爆发，[核心名场面]
        围观者反应：震惊→不敢置信→态度180度转变
        被打压者的复仇快感释放
```

## 四、伏笔系统

### 伏笔分级
- **A级伏笔（主线）**：影响全书走向，需在关键卷末回收
- **B级伏笔（支线）**：丰富世界观，在相应支线完结时回收
- **C级伏笔（调味）**：增加细节趣味，可灵活回收

### 伏笔埋设技巧
- **自然嵌入**：伏笔不能像"考试重点划线"，要自然融入场景
- **多次轻提**：重要伏笔在回收前至少提及2-3次，让读者有印象又不会忘记
- **误导性伏笔**：故意让读者往错误方向猜测，揭示真相时制造意外感
- **道具伏笔**：通过物品承载伏笔（信物、武器、信件）比抽象伏笔更易管理

### 伏笔追踪表
```
| 编号 | 伏笔内容 | 埋设章节 | 提及章节 | 回收章节 | 级别 | 状态 |

| F001 | 主角手臂上的神秘纹路 | 第1章 | 第5、12章 | 第30章 | A | 待回收 |
```

## 五、场景构建工具箱

### 场景五要素模板
每个场景（一章可含 1-3 个场景）按此结构设计：
```
场景 [编号]
├─ 目标：[POV角色在这个场景想达成什么]
├─ 阻碍：[什么阻止了他/让事情变复杂]
├─ 升级：[压力如何加大/新信息如何改变局面]
├─ 代价：[角色为了推进必须付出什么——体力/秘密/关系/资源]
└─ 钩子：[场景结尾留下什么悬念引向下一场景]
```

### 对白构建模板
写重要对话前先填：
```
对话场景：[A 与 B 的对话]
├─ A 的目的：[A 想通过这段对话获得什么]
├─ B 的目的：[B 想通过这段对话获得什么]
├─ 信息差：[A 知道但 B 不知道的 / B 知道但 A 不知道的]
├─ 权力关系：[谁占上风，是否在对话中发生翻转]
├─ 潜台词：[角色嘴上说的 vs 心里想的]
└─ 对话结果：[关系推进了/信息泄露了/冲突升级了]
```

### 场景类型速查
| 场景类型 | 核心功能 | 节奏 | 注意 |
|---------|---------|------|------|
| 冲突/对抗场景 | 推进矛盾、展示能力 | 快节奏短句 | 每回合有局势变化 |
| 情感/关系场景 | 推进感情线、揭示内心 | 中慢节奏 | 靠细节而非独白 |
| 信息/揭秘场景 | 投放关键信息或反转 | 先慢后快 | 控制信息释放节奏 |
| 日常/喘息场景 | 缓和节奏、展示人设 | 轻松 | 不超过连续2章，需埋下一个冲突种子 |
| 升级/突破场景 | 能力提升、成就达成 | 渐快→爆发 | 需仪式感，旁观者反应加分 |
