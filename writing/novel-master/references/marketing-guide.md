# 小说文案与社媒运营指南

## 一、书名设计

### 爆款书名特征
- **类型辨识度**：一看书名就知道是什么类型（"全球觉醒：我的天赋是无限叠加"）
- **悬念/好奇**：引发想看的欲望（"我死后，渣男疯了"）
- **反差/冲突**：包含矛盾元素（"废物王妃是修仙大佬"）
- **情绪共鸣**：击中读者痛点（"重生后我不再讨好任何人"）
- **角色身份**：突出主角特殊性（"全京城都在等我和离"）

### 书名结构模板
```
[身份/状态] + [反转/意外]
例：废柴少爷 + 竟是隐世强者

[时间点] + [核心行动]
例：重生第一天 + 我手撕剧本

[全员/所有人] + [态度反转]
例：全宗门都以为我废了 + 直到我拔出了那把剑
```

## 二、小说简介模板

### 50字版（封面/推荐位）
```
[主角身份] + [核心困境/事件] + [金手指/反转一句话] + [情感钩子]
```
> 例：她是被家族弃养的废材嫡女。一朝重生，随手炼出的丹药让整个大陆为之疯狂。这一世，她要站在世界之巅。

### 100字版（书城详情页）
在50字版基础上增加：
- 核心人际关系描写（1句）
- 核心冲突升级（1句）
- 风格/基调暗示（1句）

### 300字版（自媒体推广）
在100字版基础上增加：
- 世界观一句话介绍
- 1-2个精彩小情节展示
- 读者评价式推荐语或梗概概括

## 三、各平台文案调性

### 起点/番茄（男频为主）
- **调性**：直接、热血、突出爽点
- **关键词**：系统、觉醒、无敌、逆袭、碾压
- **文案结构**：困境 → 金手指 → 碾压场景预告
- **标签策略**：类型标签 + 元素标签 + 情感标签（3-5个）

### 晋江/长佩（女频为主）
- **调性**：有质感、有情绪、突出人物关系
- **关键词**：重生、宠、虐、双强、HE
- **文案结构**：人物关系简述 → 核心矛盾 → 情感走向暗示
- **标签策略**：CP类型 + 文风标签 + 情感标签

### 抖音/快手
- **调性**：口语化、冲击力、前3秒抓人
- **格式**：短视频脚本 / 图文卡 / AI配音解说
- **文案模板**：
```
"你见过[反常识场景]吗？"
"[主角名]本以为[普通期待]，没想到[戏剧反转]"
"所有人都说她[负面评价]，直到[反转事件]"
```
- **时长**：15-60秒为主，关键剧情可做2-3分钟
- **发布时间**：12:00-14:00 / 18:00-20:00 / 21:00-23:00

### 小红书
- **调性**：安利感、种草感、有共鸣
- **格式**：图文笔记 / 金句截图 / 追书日记
- **文案模板**：
```
"救命！这本[类型]文也太好看了吧 🥺"
"安利一本让我熬夜看完的小说"
"如果你喜欢[已知爆款]，那你一定会爱上这本"
标题公式：情绪词 + 类型 + 核心卖点
```
- **标签**：#小说推荐 #网文安利 #[类型]文 #追书 #书荒

### 微博
- **调性**：话题讨论、角色讨论、互动性强
- **格式**：话题tag + 角色讨论 + 投票互动
- **适合**：角色人气投票、剧情讨论、CP营销
- **文案模板**：
```
#[书名]# 今天[角色]又[关键事件]了！
[截取精彩对话/场景] 你们觉得[角色]会怎么选？
```

### B站
- **调性**：深度向、有梗、可以长内容
- **格式**：解说视频 / 人物分析 / 世界观解读
- **适合**：设定复杂、世界观宏大的作品
- **标题模板**：`[震惊/深度]这本小说的[元素]简直绝了！[具体亮点]`

## 四、社媒运营节奏

### 新书期（上架前后2周）
| 时间 | 平台 | 内容 |
|------|------|------|
| 上架前7天 | 小红书/微博 | 人设海报 + "新坑预告" |
| 上架前3天 | 全平台 | 精彩片段预告 / 金句图 |
| 上架当天 | 全平台 | 正式安利文 + 链接 |
| 上架后3天 | 抖音 | 第一个爆点剧情短视频 |
| 上架后7天 | B站/小红书 | 深度解读/角色分析 |

### 连载期（持续运营）
- 每周至少 2 条社媒内容
- 每个小高潮章节配合一条精彩片段推广
- 月度角色人气互动 / 剧情投票
- 评论区互动维护读者社群

### 完结期
- 完结感言 + 全书总结图
- 人物弧光回顾视频/图文
- 新书预告衔接

## 五、热梗融入指南

### 融入原则
- 梗要**服务剧情**，不能为了用梗而用梗
- 网络热梗的**保鲜期**通常 1-3 个月，埋入文中后注意时效性
- 用梗方式：**化用 > 直用 > 解释式引用**（越自然越好）
- 同一个梗全书最多用 2 次，避免审美疲劳

### 化用示例
```
原梗："遥遥领先"
化用："此次丹道大比，苏寻的成绩——遥遥领先于其余所有参赛者。"
（用在合适语境中，读者会心一笑但不出戏）
```
